# HikariProject

HikariProject adalah website berbasis React yang menampilkan informasi manga dan karakter anime menggunakan Jikan API. Project ini dibuat sebagai latihan frontend development dengan fokus pada penggunaan API, state management sederhana, dan UI modern menggunakan Tailwind CSS.

## Fitur

- Menampilkan daftar manga populer
- Menampilkan daftar karakter anime
- Pencarian manga dan karakter
- Detail manga dan karakter
- UI responsif menggunakan Tailwind CSS

## Teknologi

- Vite
- TypeScript
- React
- Tailwind CSS
- Node.js

## Struktur Halaman

- Home
- Manga
- Manga Detail
- Characters
- Character Detail
- Search

## API

Project ini menggunakan Jikan API:

https://api.jikan.moe/v4

## Instalasi

1. Clone repository

```bash
git clone https://github.com/rizkicahyaa/hikari-project.git
```

2. Install dependency

```bash
npm install
```

3. Jalankan project

```bash
npm run dev
```
